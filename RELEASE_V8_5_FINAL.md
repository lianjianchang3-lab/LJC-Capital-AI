# LJC Capital AI Pro V8.5 Final

## 版本定位

V8.5 Final 是 LJC Capital AI Pro 的第一个稳定使用版本，目标是每天开盘前和盘中进行快速决策查看。

## 主要能力

- 交易驾驶舱
- 自选股中心
- 持仓中心
- 执行中心
- 手机端访问
- 缓存与降级
- 统一启动入口

## 启动

桌面端：

```bash
scripts/start_desktop.command
```

手机端：

```bash
scripts/start_mobile.command
```

## 发布检查

```bash
scripts/release_check_v8_5_final.command
```
