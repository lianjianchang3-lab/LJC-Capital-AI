# LJC Capital AI Pro V3.1 RC 部署说明

## 替换文件
把本包中的以下内容复制到你的项目根目录：
- app.py
- dashboard/home.py
- engine/*.py
- config/v31_config.json

## Git 命令
```bash
git add .
git commit -m "V3.1 RC Execution Edition"
git push origin main
```

然后 Streamlit Cloud 点击 Reboot。
