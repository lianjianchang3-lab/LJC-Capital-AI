# LJC Capital AI V10.0 Professional

## 核心模块

- 市场数据中心
- Level2资金监控基础版
- 行业轮动AI
- AI投委会V4
- 自动交易计划
- 手机投委会

## 启动电脑端

```bash
python -m streamlit run apps/v10_professional.py
```

## 启动手机端

```bash
python -m streamlit run apps/v10_mobile.py --server.address 0.0.0.0 --server.port 8501
```

## 注意

本系统为投资研究辅助工具，不构成自动交易或投资建议。实盘前必须人工核验数据和风险。
